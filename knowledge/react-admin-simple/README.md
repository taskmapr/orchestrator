## React-Admin Simple Knowledge Pack

Use these briefs to give an agent high-level context on the `examples/simple` React-Admin demo without crawling the entire repository.

- `overview.md` — Application map and key moving parts.
- `data-layer.md` — Data provider decoration, fake backend, uploads, and caching.
- `posts.md` — Detailed walkthrough of the posts CRUD flows and helper components.
- `comments.md` — Moderation UI patterns, quick-create flows, and previews.
- `tags.md` — Hierarchical tag management and translation workflows.
- `users.md` — Permission-aware forms, inline editing, and validators.
- `auth-and-i18n.md` — Login roles, access control, and localization strategy.
- `layout-and-navigation.md` — Custom layout, menus, and route extensions.
- `taskmapr-integration.md` — How the TaskMapr overlay hooks into the app.

Each document highlights relevant files, hooks, and behaviors so the orchestrator can answer targeted questions or plan modifications quickly.
